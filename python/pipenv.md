## 安装
```
pip install pipenv
```

## 创建虚拟环境
```
pipenv install
```

## 进入虚拟环境（如果没有，创建虚拟环境）
```
pipenv shell
```

## 安装依赖包
```
pipenv install [package]
```

## 退出虚拟环境
```
exit
```

## 卸载
```
pipenv uninstall [package]
```

## 查看依赖
```
pipenv graph
```

## 查找虚拟目录的路径
```
pipenv --venv
```

## 删除环境
```
pipenv --rm
```
